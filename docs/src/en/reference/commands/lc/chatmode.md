# /lc chatmode <Badge type="tip" text="v0.6.0" /> <Badge type="tip" text="Paper" />

### `/lc chatmode`

| Permission                           |
|-----------------------------------|
| `lunaticchat.command.lc.chatmode` |

Displays the current chat mode.

### `/lc chatmode toggle`

| Permission                            |
|------------------------------------|
| `lunaticchat.command.lc.chatmode.toggle` |

Toggles the chat mode.
